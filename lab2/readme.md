#  Складской учёт товаров


**Приложение для ведения складского учёта** с удобным графическим интерфейсом на **Python** и **tkinter**.  
Поддерживает полный CRUD, постраничный просмотр, поиск с автоподстановкой, массовое удаление по условиям и импорт/экспорт в XML.

---

## 📌 Основные возможности

| Функция | Описание |
|---------|----------|
| **CRUD операции** | Добавление, редактирование, удаление товаров с валидацией полей |
| **Пагинация** | Настраиваемое количество записей на страницу (5/10/20/50) |
| **Умный поиск** | Автоматическая фильтрация по названию, количеству, производителю, УНП, адресу |
| **Массовое удаление** | Поиск по условиям, предпросмотр, множественный выбор, подтверждение |
| **Импорт/экспорт XML** | Загрузка (SAX) и сохранение (DOM) данных в XML-файлы |
| **Современный дизайн** | Нежно-зелёная цветовая схема интерфейса |
| **Горячие клавиши** | Быстрый доступ ко всем основным действиям |
| **Демо-данные** | Генератор 100 тестовых записей (компьютеры + обувь) |
